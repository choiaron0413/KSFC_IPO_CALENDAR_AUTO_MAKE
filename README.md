**💡 주요 기능 (Key Features)**

이 프로젝트는 단 한 번의 클릭으로 IPO(기업공개) 일정을 자동으로 조회하고 정리해주는 효율적인 도구입니다.

**원클릭 IPO 캘린더 생성:** Streamlit 앱에서 버튼 클릭 한 번만으로 38커뮤니케이션 웹사이트의 IPO 일정을 자동으로 스크롤링하여 데이터를 수집합니다.

**달력형 엑셀 보고서:** 수집된 데이터를 사용자 친화적인 달력 형태의 Excel 파일로 즉시 가공하여 제공합니다.

**로컬 저장:** 실행 파일이 위치한 폴더 내에 output 폴더를 생성하고, 생성된 Excel 파일을 해당 폴더에 저장합니다.

**🚀 향후 개발 계획 (Roadmap)**

사용자 편의성과 자동화 수준을 극대화하기 위해 다음과 같은 기능을 추가할 예정입니다.

**1. 📥 다운로드 기능 개선 (User Experience)**

현재는 결과를 로컬 output 폴더에 저장하지만, 향후 사용자가 앱 내에서 **"다운로드 버튼"**을 통해 생성된 Excel 파일을 직접 다운로드할 수 있도록 기능을 개선할 예정입니다.

**2. ⚙️ 자동화 및 보고서 배포 (Automation & Reporting)**

내 돈 주고 클라우드 서버 굴릴순 없고... 회사 외부 서버도 없어서...
나중에 서버 환경이 구축되면, 아래와 같이 완전 자동화된 보고서 배포 시스템을 구현할 계획입니다.

**주기적 수행:** 서버에 프로그램을 배포하여 정기적/주기적으로 IPO 일정을 스크롤링하고 캘린더를 생성합니다.

**SMTP 메일 발송:** 생성된 Excel 파일을  SMTP(Simple Mail Transfer Protocol)를 연결하여 자동으로 담당자 회사 이메일로 발송하는 시스템을 구축할 예정입니다.

**3. 👀 하나의 실행파일로 빌드 방법**

pyinstaller --onefile --additional-hooks-dir=./hooks --name "KSFC_IPO_Calendar_Auto_Make" --add-data "welcome_app.py;." --add-data "ipo_search.py;." --add-data "calendarUtils.py;." --hidden-import "streamlit.runtime.scriptrunner.magic_funcs" --hidden-import "streamlit.web.cli" --hidden-import "streamlit.web.bootstrap" KSFC_중복청약_IPO_캘린더.py --clean 

**4. 👴 기존 IPO 캘린더 소스와 다른점**
 1) 최초 개발자인 이용준 과장님 소스에서 deprecated 된 코드가 꽤 있어서 이를 현재 파이썬 룰에 맞춰 개발했습니다.
 2) 날짜 입력 포맷이 정해져있었는데 포맷을 안지키면 빈 엑셀이 나와서, 이거를 그냥 딸깍으로 바꿨습니다.
 3) 주피터 베이스 파일은 모두 제거하여 개발자 혼선을 제거하였습니다.
 4) 주요 기능은 다른 파일로 분리하였습니다. 다만, 깔끔하게 분리는 아직 못했습니다.
